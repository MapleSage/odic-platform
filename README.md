# odic-platform

Implementation repository for the ODIC Enterprise Intelligence Platform.

## What this repo now contains
- `apps/shell` — primary React shell for workspace-driven UX
- `packages/*` — shared UI/system libraries
- `services/*` — API, connectors, and orchestration service skeletons
- `docs/` — engineering and architecture notes

## Product rule
This repo implements the blueprint. It does not redefine product UX.

## Priority implementation order
1. Shell + routing
2. Workspace core primitives
3. Search + Organization + GIA surfaces
4. Connector and graph services
5. Reporting and dashboard generation

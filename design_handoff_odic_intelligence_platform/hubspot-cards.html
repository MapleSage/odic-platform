BUILD NOTE
These are visual specs for HubSpot's CRM Cards integration (one card per object type: Company, Contact, Deal). Each card is static here with sample data -- once the backend/API layer is ready, the same layout renders live values via the CRM Cards API (token fields, a title, and up to a few actions per card). Card content should stay short: 3-5 data points plus one primary and one secondary action, matching HubSpot's own card density conventions.

# OpenClaw Engineering Brief

## Mission
Build the ODIC Enterprise Intelligence Platform.
Do not build isolated dashboards.
Do not clone ZoomInfo.
Do not clone Microsoft Fabric.
Do not clone Bloomberg.
Build a modern Enterprise Intelligence Platform inspired by them.

## Mandatory Stack
- React
- TypeScript
- Vite
- Fluent UI v9
- TanStack Router
- TanStack Query
- React Flow
- D3
- AG Grid Enterprise
- Power BI Embedded
- Azure Maps
- Azure Entra
- Azure Fabric
- Azure AI Foundry
- Microsoft Graph
- OpenSearch
- FastAPI
- Rust APIs

## Product Rules
- Everything is workspace-driven.
- All AI is exposed as GIA only.
- HubSpot is an operational connector, not source of truth.
- Industry modules are pluggable.
- Reports are first-class objects.
- Dashboards are generated, not manually built.

## Definition of Done
The platform must feel like an Enterprise Intelligence Operating System, not a CRM, BI tool, or dashboard app.
